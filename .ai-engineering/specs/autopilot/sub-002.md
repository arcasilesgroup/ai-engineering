# Sub-Spec 002: Stack Operation Tests

## Scope

Create `tests/unit/test_stack_operations.py` with >= 6 tests covering `add_stack`, `remove_stack`, and `list_status`.

## Exploration

- `add_stack(target, stack)` -> validates against `get_available_stacks()`, checks duplicate, writes manifest, logs audit
- `remove_stack(target, stack)` -> checks presence, writes manifest, logs audit. No guard against removing last stack
- `list_status(target)` -> delegates to `_load_config()`, returns ManifestConfig
- All raise `InstallerError` on failures
- Need to mock: `append_ndjson`, `get_repo_context`, `get_git_context` (audit side-effects)
- Fixture: tmp_path with `.ai-engineering/manifest.yml` containing `providers.stacks`
